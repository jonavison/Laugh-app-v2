# Geist (subtitle default)

Subtitles use **Geist Sans SemiBold** from [Vercel’s Geist family](https://vercel.com/font) (SIL Open Font License 1.1).

- `Geist-SemiBold.otf` — bundled for in-app rendering
- `LICENSE.txt` — full OFL text

To refresh the font file, download the latest release from [vercel/geist-font](https://github.com/vercel/geist-font/releases) and copy `geist-font/Geist/otf/Geist-SemiBold.otf` here.
