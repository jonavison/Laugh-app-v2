# Documentation Policy

This repository requires documentation updates with every meaningful change.

## Required on Every Change

- Update `CHANGELOG.md` with what changed and why.
- If codec/container/media behavior changed, update `SUPPORT.md`.

## Done Criteria

A change is not considered complete until:
- code compiles/tests pass (as applicable), and
- documentation reflects the new behavior.

## Scope

Applies to:
- feature work
- bug fixes
- behavior/config changes
- compatibility support updates
